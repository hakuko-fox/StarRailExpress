# 废弃监狱七日夜

## 人员配置
### 杀手阵营
- 两个狂魔（特殊杀手）固定刷新
- 四个杀手（普通） 4 / 50
- 一个毒师 固定刷新一个
### 平民阵营
- 两个军人（平民）2 / 50 固定刷新
- 一个厨师 固定刷新
- 两个心理学大师（pua被抓） 2 / 50
- 四个锁匠（撬锁被抓） 4 / 50
- 剩余（34）（平民）

## 人物功能

### 杀手阵营

#### Npc狂魔
- 白天为旁观模式，夜晚在自己房间变为狂魔，拥有速度2
- 被枪杀会眩晕一个晚上
- 第一次建议录视频的摄像头当

#### 杀手
- 开局与正常罪犯无差别，需要完成任务
- 晚上觉醒杀手，有一把刀和撬棍
  - 刀一晚上只能开三次
  - 撬棍只能使用一次
- 杀掉一个人之后可以选择申请援助，在第二天晚上获得开锁器或手枪
  - 开锁器可以使用3次一次性
  - 枪可以隔着玻璃杀人一次性
- （你们无需吃饭，可以啃食尸体）

#### 毒师
- 可以制作被污染食物：肉，黑面包，玉米糊糊（要不改成可以污染食材，导致厨师制作出来含毒？）
- 第一天可做五个，后面每天只能有一个
- 杀死五人后解锁污染水源选项
- 污染食物需要夜里放入食物箱子里面

### 好人阵营
#### 军人（保安？）
- 拥有一个警棍，一把手枪，一天只有一发子弹
- 白天不可枪杀别人，否则降级为罪犯（平民）
- 投票完成后警察会对被票人有高亮，此时警棍可以俩下敲死罪犯

#### 厨师
- 白天需要在食堂工作，为明天所有人制作食物。
- 检查水源，警惕里面食物是否被掉包（设置一个专门食物箱子）
- 食物种类有：玉米糊糊（可使用原版的各种堡，增加效果还可以使用迷之炖菜），黑面包（可用原版面板代替），肉（可使用原版物品）
- 水源检查需要占用10个食物位（不理解）
- 厨师最多能做40个食物（限制制作数量？）
- 食堂一开始会自带50份食物（第一天自带，可使用原版箱子）
- 厨师的工作可以帮他回san（做一个食物回满san值）

#### 心理学大师
- 可以以扣80%san为条件观察一个人是否杀过人

#### 锁匠
- 可以用25%做一个维修工具来修复被翘过锁的门（只能修复撬锁器损坏的门）

#### 普通罪犯
- 白天需要吃饭，听讲会，上厕所
- 发现尸体可以报告进入公告会中，有着票人权力

## 综合背景
> 星星历8440年，你们作为流民无处可归即将饿死，一个善良a国军官，以普通罪犯名义将你们抓捕，进行安顿下来七天混到a国国民认证。可是一路上你们时不时就少人，从最开始60多人逐渐变成最后48人，但你们太饿了，也因为战争已经麻木了，认为不过是走散了。在一个傍晚你们到了这个曾今因为一些原因被废弃监狱，换来片刻的安宁。

## 规则怪谈内容
### 食堂牌子
1. 你们登入监狱后，属于下午，第一天早上只有2分钟，需要快速吃饭
2. 食物是有限的，请不要浪费粮食
3. 夜晚食堂相对安全，实在没地方去可以来食堂过夜
4. 食堂食物不一定安全

### 宿舍楼牌子
1. 在自己房间里面是相对安全的，不要随便窜门
2. 宿舍楼夜里有怪物出没请勿出门
3. 保持一个80%san值，否则你的房间将不认识你
4. 看到尸体请报告

### 警察宿舍
1. 保证人员安全，枪是最后的保障，而不是制裁手段
2. 怪物白天会死亡，请找到流民里面藏着的吃人的东西
3. 恶魔无法杀死，但是枪可以暂时驱逐它
4. 你们永远是安全的，只要你不出门
